# SplitMates

Web application that is used for calculating expenses on trips with your friends.

## Description

SplitMates is a web application designed to simplify group expense tracking during shared experiences like vacations, road trips, or events with friends. It allows users to create a group, invite members, and log shared expenses by selecting who paid and who was involved. At any point, the app calculates who owes what, making it easy to settle up at the end of the trip. Built with React for a responsive frontend and Firebase for secure authentication and real-time data storage, SplitTrip ensures that managing money with friends is clear, fair, and hassle-free. The goal is to reduce the mental load of tracking shared costs and prevent misunderstandings, so users can focus on enjoying their trip. Future updates will include features like automatic debt simplification, notifications, and integration with payment providers like PayPal for one-click settling.
